# SentraAI 🚀

AI-powered Customer Complaint Dashboard

## Features
- NLP-based categorization
- Sentiment analysis
- Duplicate detection
- AI-generated responses
- SLA tracking
- Escalation management
- 360° complaint view

## Tech Stack
- Node.js + Express
- MongoDB Atlas
- React.js
- OpenAI API
- Transformers

## Setup

### Backend
cd backend
npm install
npm start

### Frontend
cd frontend
npm install
npm start
